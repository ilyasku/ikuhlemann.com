Welcome!

I studied biophysics and complex systems in my master's at Georg-August-University Göttingen. When I started studying I thought physicists would look at stars and shoot particles at other particles all day to derive the existence of the universe from it, and I wanted to do the same. But during my studies I became more and more attracted to programming, computational neuroscience and robotics.

I want to use this website for three things:
  • Learn/improve my HTML, CSS and JavaScript.
  • Present larger software and hardware projects I pursued or will pursue, e.g. from one of my student assistant jobs (RunBot) or my master's thesis (CompoundPye) which I'm currently working on.
  • Create an unified up-to-date portfolio in a way I like, instead of updating it on all the other platforms like ResearchGate, LinkedIn, ...
    But so far this website is still in its infancy!

This Website

I want to learn HTML, CSS and JavaScript (and PHP and MySQL in the long run), starting by building this website and hosting it on my own. It seems to be the best way to build interfaces usable with most OS and devices, e.g. if I want to control a Raspberry Pi in a local network.
One related thing that made me happy after learning the very basics of HTML was reveal.js, which I think I'm going to stick to for future presentations. It should be able to display videos, gifs and whatever you want to show in your presentation on most operating systems and browsers. I hope.

But seriously, how could I live without the comfort of hyperlinks so far?
